# 📸 Camera-AI: Object Localization

## 📖 Overview
An independent computer vision venture focused on real-time object detection and dataset labeling. This application processes live camera feeds to identify objects and overlay accurate bounding boxes, bridging raw camera data with visual classification.

## 🛠️ Focus Areas
* **Algorithm Implementation:** Working with OpenCV for live frame processing.
* **Visual Output:** Drawing clear, responsive bounding boxes and labels on video streams.
* **System QA:** Testing localization accuracy across varying lighting and environment conditions.
